# Advanced Search Settings

Fine tune the search capabilities.

kernl(wp) leverages a premium WP Plugin called Relevanssi Pro. This enhances the default search by:

  • Indexing
    • PDFs
    • Custom fields
    • Taxonomy terms
  • Fuzzy search matching
  • Configurable stop words
  • Adjust weight of certain terms
  • Add synonyms for search terms

TIP

Please reach out to provost-digital@northeastern.edu for initial setup.
