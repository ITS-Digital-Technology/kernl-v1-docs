# WYSIWYG

The WYSIWYG editor (What You See Is What You Get) allows you to format text and media before publishing it.

# Overview

At its most basic level, the WYSIWYG is the tool in which you will compose and edit most of your website's content including, but not limited to, posts, pages, and other types of content. In depth instructions on how to use the WYSIWYG to create pages and post are in Core Views

Admin View

Here you can add a title, customize the banner, and compose, arrange, and format the body of the content.
