# Events

Events are a type of post that can be organized into various types of calendars.

# Create an Event

Events and Posts share many of the same input fields and behaviors. Creating an event is just like a post or a page. Adding a category is important especially if you have multiple events on the same day. How to createPosts may be a helpfull refresher.
